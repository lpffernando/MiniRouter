# Scripts

Helper scripts for development and operations. Not included in the Docker image.

- `today.mjs` — Daily dashboard (queries SQLite directly)
- `local-smoke.mjs` — Local smoke test (`node scripts/local-smoke.mjs`)
- `cache-usage-probe.mjs` — Probe cache usage patterns
- `stream-cache-probe.mjs` — Probe stream cache behavior

> One-time audit scripts have been moved to `archive/`.
