# Tools

Model score import and maintenance utilities. Not included in the Docker image.

- `import-llm-stats.mjs` — Import model scores from LLM Stats API
- `import-superclue-scores.mjs` — Import SuperCLUE benchmark scores
- `scope-model-scores.mjs` — Audit model scores in SQLite vs seed data
- `score-utils.mjs` — Shared scoring utilities
- `score-utils.test.mjs` — Tests for score-utils
- `model-benchmark-aliases.mjs` — Model ID aliases for benchmark lookup
- `superclue-model-aliases.mjs` — Model ID aliases for SuperCLUE lookup
